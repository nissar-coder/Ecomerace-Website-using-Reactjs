# A modern, fast, and fully responsive e-commerce web application built with React.js. Designed to provide users with a smooth and enjoyable online shopping experience.

✨ Features

🛍️ Product listing with clean UI
🛒 Add to cart / Remove from cart
💰 Dynamic cart total calculation
📱 Fully responsive on all devices
⚡ Fast performance with Vite
🎨 Clean and modern design



🚀 Tech Stack

React.js - Frontend framework
React Router DOM - Page navigation
Tailwind CSS - Styling
useState / useContext - State management
Vite - Build too
